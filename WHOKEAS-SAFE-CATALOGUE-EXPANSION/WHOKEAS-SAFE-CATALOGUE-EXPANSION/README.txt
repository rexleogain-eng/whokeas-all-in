WHOKEAS ALL IN — SAFE CATALOGUE EXPANSION

WHAT IT INSTALLS
- Permanent cleanup of the six original design-trial products
- Safe queued CJ discovery and importing
- One-at-a-time CJ requests with throttling retries and exponential backoff
- Duplicate detection by CJ ID, exact product name and normalized fingerprint
- Category targets for a balanced 170-product initial catalogue
- Approximately 30% gross-margin pricing
- Admin control page at /admin/catalogue
- Daily Vercel cron and an optional GitHub Actions schedule

INSTALLATION
1. Stop npm run dev with Ctrl+C.
2. Right-click the downloaded ZIP and choose Extract All.
3. Open the extracted WHOKEAS-SAFE-CATALOGUE-EXPANSION folder.
4. Double-click RUN-WHOKEAS-SAFE-CATALOGUE-EXPANSION.cmd.
5. Keep the PowerShell window open until the final result appears.

AFTER INSTALLATION
1. Start locally: npm run dev
2. Open: http://localhost:3000/admin/catalogue
3. Click Fill store safely.
4. Complete WHOKEAS-AUTOMATION-SETUP.txt on your Desktop for unattended runs.

The installer creates a Desktop backup and a full log. It never copies or prints your API keys.
