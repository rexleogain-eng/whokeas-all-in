WHOKEAS ALL IN — Classic Admin Suite

INSTALLATION
1. Stop npm run dev with Ctrl+C.
2. Extract this ZIP completely.
3. Open the extracted folder.
4. Double-click RUN-WHOKEAS-CLASSIC-ADMIN-SUITE.cmd.
5. Wait for the successful production-build message.
6. Start the project again with npm run dev.
7. Open http://localhost:3000/admin

WHAT CHANGES
- Adds a new executive overview page.
- Adds a classic dark, ivory and muted-gold admin shell.
- Redesigns Orders, Product Control, CJ Sourcing and Admin Login.
- Adds product statistics, product search and status filtering.
- Keeps all existing APIs, database logic and customer storefront unchanged.
- Creates a Desktop backup before replacing admin files.
