WHOKEAS STAY-OPEN INSTALLER

1. Stop npm run dev with Ctrl+C.
2. Right-click the ZIP and choose Extract All.
3. Open the extracted WHOKEAS-STAY-OPEN-30-PUBLISH folder.
4. Double-click RUN-WHOKEAS-STAY-OPEN.cmd.
5. The PowerShell window is forced to remain open.
6. On failure, a full log is saved on your Desktop as:
   WHOKEAS-30-PUBLISH-YYYYMMDD-HHMMSS.log

Do not run the CMD while it is still inside the ZIP.
